# murmuration

Generative flock art that lives on GitHub Pages with no server — and no memory
except the visitors themselves. Every visitor grows a unique flock from a seed;
visitors online at the same time see their flocks merge, and departed visitors'
birds survive by being carried in other people's browsers.

**Status: planning.** See [PLAN.md](PLAN.md) for the architecture and roadmap,
and the deciduous decision graph (`deciduous serve`) for why each choice was made.
